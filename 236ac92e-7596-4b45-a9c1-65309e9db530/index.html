Churn Rate (чорн, баунс-рейт?, відтік)) — показник відтоку клієнтів / користувачів.

Навіщо потрібно знати Churn Rate

Як рахувати Churn Rate

Churn Rate = кільк-ть клієнтів, що пішли / загальну кільк-ть клієнтів * 100%

(приклад для саас? чи потрібен в інших сферах)

Для SaaS — відмінив підписку / не продовжував підписку

Для Ecom — дані про частоту покупок, сегменти (RFM-аналіз) — коли стає зрозуміло, що клієнт більше не буде купувати (не купував більш, ніж півроку)
